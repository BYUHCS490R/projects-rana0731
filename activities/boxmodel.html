Box Model

1. What is the difference between block tags and inline tags?

Block Box - Create new box line on screen. Normal flow (Default) take up 100% full width. (div, p, h1-h6, ul, li)

Inline Box - Stay in the same box line. (span, img, strong, em, a)

2. What is the margin, border, and padding?

Margin is outside space

Border is inside margin. It is line around padding

Padding is inside box. It is inside space

3. What are units of measurement you can use in CSS?

px, cm, mm, in, pt, pc are Fixed size (does not change with screen)

em, rem, %, vw, vh, vmin, vmax are Adjusts depending on parent or viewport size

4.When would you use a figure tag?

Use the "figure" tag when you have self-contained content like an image, chart, code snippet, or diagram with an optional caption that explains it.
