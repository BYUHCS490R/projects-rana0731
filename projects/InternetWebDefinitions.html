4.ASCII and Unicode

Both translate characters to numbers for programing. ASCII encodes English letters digits symbols using 7 bits, while Unicode encodes characters from all languages using 16-32 bits


5.What is UTF-8

Unicode Transformation Format. most common, encodes all charactar in unicode using 1-4 bytes each which means translate characters into number.


6.URL

Uniform Resource Locator. it is address telling browser how and where to go page on browser (with slash /)


7.Domain Name

8.HTTP

HyperText Transfer Protocol. Transfer web pages and data over the Internet, itensures that data is sent, received correctly between computers. But it is not secure.


9.HTTPS

A secure version of HTTP, can protect personal info or data


10.IP Address

Internet protocol address. Every computer has numbers such as 192.0.2.1 connected to internet


11.DNS

Domain name system. It transfer domain name to IP address. It let us use easy name instead of bunch of numbers


12.W3C Recommendations

World Wide Web Consortium. is website standard. It is guidance how to make webs are consistent, compatible and works same on all browsers. Also html, css, xml etc web technologies passed through the review and testing process, they are W3C Recommendations.


13.XML, XHTML, HTML, HTML5

are W3C Recommendations, meaning they are official web standards
XML store and transport data, while XHTML HTML HTML5 create web page XML Extensible Markup Language. key: It allows users to create own tag
XHTML Extensible HyperText Markup Language. Written as XML version of HTML, key: all tags has closing slash and need to be nested.
HTML Hypertext Markup Language. key: Standard/core language of the Web, include elements such as heading, paragraph, table, h1-h6 font.
HTML5 key: a modern upgrade version of HTML which has audio, video, canvas (graphic) tags.
