1. Please describe the parent-child relationship in HTML and why it becomes essential when using flex and grid.

Both Flexbox and CSS Grid are layout models that control how child elements are arranged inside their parent container.

2. What are the two main differences between CSS Grid and CSS Flex?

    Grid
  • Two dimensional /both rows and columns/
  • Layout First
  • Creating complex layouts with rows and columns
    Flex
  • One Dimensional /a single axis/
  • Content First
  • Aligning items in a row or column
