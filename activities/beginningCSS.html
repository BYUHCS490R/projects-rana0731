Beginning CSS

1. Explain the difference between inline, embedded, and external CSS.

Inline CSS

Write directly inside p tag using the style attribute. style="color: blue;"

Embedded (Internal) CSS

Write inside the style tag in the head section of an HTML file. p {color: blue;}

External CSS

Write in a separate .css file p {color: blue;} and linked to the HTML using link tag in head section.

2. Explain why you should only use external CSS.

Using separate CSS will be organized. CSS file for only style and HTML file only for content

3. What is a selector used for?

A CSS selector is used to target HTML elements such as tags (p), id or class. And will style only that selected HTML elements.

p { color: blue; } Style all p elements.
