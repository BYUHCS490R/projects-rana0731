CSS Selector

1. List out the different ways to select an item.

Element(tagname) selector, ID selector, Class selector, Universal selector (*), Child selector

2. What is the difference between an ID and a Class?

ID can be used for only one element and must be unique title

h1 id="unique-title">Welcome to My Page /h1

Class can be used by multiple elements.

p class="highlight">This paragraph is highlighted. /p
p class="highlight">This is another highlighted paragraph. /p

3. What is the difference between a SPAN tag and a DIV tag?

  • DIV Block-level element.Starts on a new line, takes full width.Used for larger sections/containers.
    div class="container" > This is a section. /div
  • SPAN Inline element.Does not start a new line, fits within text.Used for styling specific parts of text.
    p This is a span class="highlight" > highlighted word. /p

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV. Used only for grouping and styling content. While header main footer are structural(section) tags.

5. What are the different ways to select a color?

  • Hex Codes (A # and then 6 hex characters. Example #F5A4D3)
  • Hex Short Codes (A shortcut with only 3 Hex characters. Example #FDA = #FFDDAA)
  • Color Names
  • RGB Values rgb(red, green, blue)
  • RGBA Values rgba(red, green, blue, alpha)
  • HSL Values hsl(hue, saturation, lightness)
  • HSLA Values hsla(hue, saturation, lightness, alpha)
